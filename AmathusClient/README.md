# Amathus Client

Amathus Android and iOS client in Flutter

## Build

Android:

```sh
flutter build apk --debug
```

iOS:

```sh
flutter build ios --debug
```

## Emulators

Create emulators in Android or iOS.

List emulators:

```sh
flutter emulators
```

Launch an emulator:

```sh
flutter emulators --launch <your-emulator-id>
```

## Run

Run in an emulator:

```sh
flutter run --debug -d <your-emulator-id>
```
